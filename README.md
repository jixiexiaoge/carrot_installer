# Carrot 远程安装工具

OpenPilot/Carrot 设备的 Windows 远程安装工具集，支持通过 SSH 自动部署和管理 OpenPilot 分支。

## 功能特性

- 🚀 **远程安装**: 通过 SSH 远程安装 OpenPilot 到 Comma 设备
- 🔍 **自动扫描**: 自动扫描局域网中的 Comma 设备并安装
- 🔐 **安全连接**: 使用 SSH 密钥认证，确保连接安全
- 💾 **智能备份**: 自动备份现有版本，安装失败时自动回滚
- 🌐 **多工具集**: 提供扫描、连接、安装等多种实用工具

## 工具说明

### 1. install.bat - 手动指定 IP 安装
手动输入设备 IP 地址和分支名称进行远程安装。

**使用场景**: 当你已知设备的 IP 地址时使用

**使用步骤**:
1. 双击运行 `install.bat`
2. 输入设备 IP 地址（如：192.168.1.100）
3. 输入要安装的分支名称（如：master）
4. 等待安装完成，设备将自动重启

### 2. install_auto.bat - 自动扫描并安装
自动扫描指定网段内的所有 Comma 设备，找到后自动安装。

**使用场景**: 当你不确定设备 IP 或网络中有多个设备时使用

**使用步骤**:
1. 双击运行 `install_auto.bat`
2. 输入网络前缀（如：192.168.0，会自动扫描 192.168.0.1-255）
3. 输入要安装的分支名称
4. 等待扫描和安装完成

### 3. ssh_connect.bat - SSH 连接工具
快速连接到 Comma 设备的 SSH 终端。

**使用场景**: 需要手动登录设备执行命令时使用

**使用步骤**:
1. 双击运行 `ssh_connect.bat`
2. 输入设备 IP 地址
3. 成功连接后可以执行各种 Linux 命令

### 4. scan_ssh_open.bat - 网络扫描工具
快速扫描指定网段内开放了 SSH (端口 22) 的设备。

**使用场景**: 查找网络中所有开放 SSH 的设备

**使用步骤**:
1. 双击运行 `scan_ssh_open.bat`
2. 输入网络前缀（如：192.168.0）
3. 等待扫描完成，会列出所有开放 22 端口的设备

### 5. find_comma.bat - 快速查找 Comma 设备
使用并行扫描技术快速定位 Comma 设备。

**使用场景**: 快速在网络中找到 Comma 设备的 IP 地址

**使用步骤**:
1. 双击运行 `find_comma.bat`
2. 输入网络前缀
3. 自动并行扫描并验证 Comma 设备

## 前置要求

### 必需文件
确保以下文件存在于工具目录中：

- ✅ `plink.exe` - PuTTY 的命令行 SSH 客户端
- ✅ `id_rsa.ppk` - Comma 设备的 SSH 私钥文件

### 系统要求
- Windows 7 及以上版本
- PowerShell 3.0+ (Windows 8+ 自带)
- 网络连接（设备和电脑在同一局域网）

## 安装流程说明

安装程序会执行以下步骤：

1. **备份当前版本**
   ```
   openpilot → openpilot_stock
   openpilot_stock → openpilot_stock_prev
   ```

2. **克隆新分支**
   ```
   从 https://gitcode.com/jixiexiaoge/openpilot.git 克隆指定分支到 openpilot_new
   ```

3. **部署新版本**
   ```
   openpilot_new → openpilot
   创建 /data/continue.sh 启动脚本
   ```

4. **重启设备**
   ```
   自动重启以启动新安装的版本
   ```

### 安全机制

- ✅ **失败回滚**: 如果安装失败，自动恢复之前的版本
- ✅ **磁盘检查**: 安装前检查磁盘空间
- ✅ **版本备份**: 保留前两个版本以便回滚
- ✅ **完整性验证**: 确保克隆成功后才进行替换

## 使用注意事项

### ⚠️ 重要提示

1. **网络连接**
   - 确保 Comma 设备已连接到 Wi-Fi
   - 电脑和设备必须在同一局域网
   - 首次连接时可能需要接受主机密钥（输入 `y` 确认）

2. **SSH 密钥**
   - `id_rsa.ppk` 必须与设备匹配
   - 默认用户名为 `comma`
   - 不要随意修改密钥文件

3. **安装时间**
   - 完整安装过程约需 5-15 分钟
   - 取决于网络速度和设备性能
   - 请勿在安装过程中中断连接

4. **分支名称**
   - 确保输入正确的分支名称
   - 分支必须存在于源仓库中
   - 常用分支：`master`, `devel` 等

### 🔧 故障排除

**问题**: SSH 连接失败
```
解决方案:
1. 检查设备 IP 是否正确
2. 确认设备已连接 Wi-Fi
3. 验证 id_rsa.ppk 文件是否正确
4. 尝试接受主机密钥（输入 y）
```

**问题**: 克隆失败
```
解决方案:
1. 检查网络连接
2. 确认分支名称正确
3. 检查设备磁盘空间是否充足
4. 程序会自动回滚到之前的版本
```

**问题**: 找不到设备
```
解决方案:
1. 确认设备已开机并连接 Wi-Fi
2. 检查网络前缀是否正确
3. 尝试使用 find_comma.bat 快速定位
4. 手动检查设备的 IP 地址
```

## 常见问题 (FAQ)

**Q: 安装会删除我的数据吗？**
A: 安装程序只操作 `/data/openpilot` 目录，会自动备份现有版本。个人数据和设置通常不受影响，但建议提前备份重要数据。

**Q: 可以安装第三方分支吗？**
A: 目前只支持 `https://gitcode.com/jixiexiaoge/openpilot.git` 仓库的分支。如需安装其他仓库，需要修改脚本中的 Git 仓库地址。

**Q: 安装失败后怎么办？**
A: 安装程序有自动回滚功能，失败时会自动恢复之前的版本。设备仍可正常启动。

**Q: 支持 Mac 或 Linux 吗？**
A: 这些工具专为 Windows 设计。Mac/Linux 用户可以直接使用 ssh 和 git 命令手动安装。
